# 语法

qas 的执行流程使用 yaml 描述，完全遵循 [yaml](https://yaml.org/spec/1.2.2/) 的描述语法

一个最简单的可执行的完整 case 如下：

```yaml
name: hello-world

ctx:
  shell:
    type: shell

case:
  - name: EchoHelloWorld
    step:
      - ctx: shell
        req:
          command: echo hello world
        res:
          exitCode: 0
```
