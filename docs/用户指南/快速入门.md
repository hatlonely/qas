# 快速入门

## 安装

### 源码安装

```shell
git fetch --depth 1 --branch master https://github.com/hatlonely/qas.git && \
cd qas && \
python3 setup.py install
```

## 快速入门

1. 创建一个 case 文件 `helloworld/ctx.yaml`

```yaml
name: hello-world

ctx:
  shell:
    type: shell

case:
  - name: HelloWorld
    step:
      - ctx: shell
        req:
          command: echo -n hello world
        res:
          exitCode: 0
          stdout: hello world
```

2. 执行测试

```shell
qas --test helloworld
```
