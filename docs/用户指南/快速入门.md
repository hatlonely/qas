# 快速入门

1. 创建一个 case 文件 `helloworld/ctx.yaml`

```yaml
name: hello-world

ctx:
  shell:
    type: shell

case:
  - name: HelloWorld
    step:
      - ctx: shell
        req:
          command: echo -n hello world
        res:
          exitCode: 0
          stdout: hello world
```

2. 执行测试

```shell
qas --test helloworld
```
